### 🚀 A qué me dedico

Diseño e implemento **automatizaciones con IA** y **aplicaciones web a medida** para empresas. De principio a fin: hablo con el cliente, modelo el problema, lo construyo y lo dejo corriendo en producción.

- 🤖 **Automatización (n8n + IA):** workflows, chatbots y asistentes que conectan correo, Telegram, hojas, bases de datos y modelos de lenguaje.
- 🧾 **SaaS y backends:** APIs en FastAPI + PostgreSQL, multi-tenant, autenticación, despliegue con Docker.
- 💻 **Desarrollo web:** landings, webs corporativas y CRMs con React/Vite, diseño cuidado y conexión a backend real.
- 🧠 **Integración de IA:** OCR, RAG con vectores (Pinecone), generación de contenido y routing de modelos (Claude, Gemini) vía OpenRouter.
